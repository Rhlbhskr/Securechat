1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/secure-chat-app.git
   cd secure-chat-app
   ```

2. **Install dependencies**

   - For frontend:
     ```bash
     cd client
     npm install
     npm run dev
     ```

   - For backend:
     ```bash
     cd server
     npm install
     node index.js
     ```

3. **Access the app**
   - Frontend at: `http://localhost:5173`
   - Backend running at: `http://localhost:5000`
